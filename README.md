# BTTM Translate

A google translate alternative powered by Brain.js! :D

## Requirements
- NodeJS
- NPM

## Usage
```bash
node index.js <lang from>-<lang to> <sentence>
```

### Example

```bash
node index.js nl-en Hoi!
```

## Training
```bash
node train.js <lang from>-<lang to>
```

### Example

```bash
node train.js nl-en
```
